# LLM の仕組み体験アプリ

武蔵野美術大学の学生向けに作った、ブラウザだけで完結する小さな教材です。
「LLM（大規模言語モデル）は、内部で何をしているのか」を 3 つの視点で体験できます。

- ① **次の単語をあてる** — 次トークン予測と「温度（ランダムさ）」
- ② **言葉を分解する** — トークン化（日本語と英語の違い）
- ③ **意味の地図** — 単語をベクトルとして並べた散布図

> これは仕組みを直感で理解するための簡略化されたデモです。実際の LLM はニューラルネットワークを使い、はるかに大規模です。

## ローカルでの開き方

特別なビルドもサーバも要りません。

```bash
# このリポジトリを開いて
open index.html
# もしくは index.html をダブルクリック
```

ブラウザで `file://...` として開くだけで動きます。`fetch` を使わず、データはすべて `data.js` に同梱しているためです。

## ファイル構成

```
index.html  // 3 タブの UI 骨格
style.css   // スタイル
app.js      // タブ切り替え・描画・温度ロジック
data.js     // 見本文・トークン例・意味の地図のデータ
README.md   // このファイル
```

外部ライブラリは使っていません。フレームワークも不使用、素の HTML/CSS/JavaScript です。

## データの差し替え方

- **タブ 1**：`data.js` 内 `sentences` に、見本文＋各ステップの候補テーブルを追加してください。各 `candidates` の `prob` の合計は必ず `1.0` にします（温度 1.0 のときの基準分布として扱われます）。
- **タブ 2**：`tokenization` に例文とトークン分解結果を追加します。`id` は擬似的な番号で構いません（一貫していれば OK）。
- **タブ 3**：`meaningMap.words` に単語と座標（0〜100 の x, y）を追加します。同じカテゴリは固まり、別カテゴリは離れる位置に置くと、地図として読みやすくなります。
